Motor control Contrôle de moteur

School project
Projet scolaire

During my 4th year, we had to realize the servo-control of a synchronous motor with a vector control. During this project I learned how to build PID controller and create command interfaces using Qt. The information were transmitted from the computer to the STM32-Nucleo microcontroller via Zigbee.

Pendant ma 4ème année à l'ENIB, j'ai dû réaliser le contrôle d'un moteur synchrone avec des commandes vectorielles. Dans ce projet, j'ai développé mes connaissances en régulateur PID et créer des interfaces de commandes avec Qt. Les données étaient transférées entre l'ordinateur et un microcontrolleur STM32-Nucleo avec Zigbee.

Biken Software

School project
Projet scolaire

During our 3rd year at ENIB, I developed with friends a software to generate random bike routes in France. This software uses a path-finding algorithm (A*) to create the best appropriate path. You can choose the distance you want to travel or you can simply define a starting point and a destination point. This uses the OpenStreetMap data. You can check the project demo on YouTube.

Pendant ma 3ème année à l'ENIB, j'ai développé avec des amis un logiciel pour générer des itinéraires de vélo en France. Ce logiciel utilise un algorithme de recherche de chemin (A*) pour déterminer le chemin le plus approprié. L'utilisateur peut choisir la distance souhaitée ou simplement un point de départ et d'arrivée et le logiciel construit l'itinéraire. Ce projet utilise les données Open source de OpenStreetMap. Vous pouvez voir une démonstration du projet sur YouTube.

Robot programming Programmation de robot

School project
Projet scolaire

In this project we had to code the robot. We were using a real-time operating system. The goal of the project was to distribute the different tasks and to manage the priorities in the interruptions created by the different sensors. We also had to create a control interface and a tracking of a person via image/color recognition by the camera.

Dans ce projet, j'ai dû coder un robot contrôlé par une carte STM32. Nous avons utilisé un système d'exploitation temps-réel RTOS. Le but du projet était de répartir les tâches et de gérer les interruptions physiques pour traiter les informations en temps réel. J'ai développé une interface de contrôle pour diriger le robot depuis un clavier et utilisé des algorithmes de reconnaissance d'image pour permettre au robot de suivre une personne ou une couleur particulière.

Sensor network Réseaux de capteur

School project
Projet scolaire

In this project I had to learn how the CAN, I2C and SPI buses worked since all three protocols were used in the different sensors. I had to manage the transmission of information from five sensors (brightness, proximity, wind speed, humidity and gyroscope) simultaneously via a single CAN bus and display the data in real time on an interface.

Ce projet consistait à connecter différents capteurs à un même réseau CAN et récupérer les données. J'ai appris à utiliser les protocoles CAN, I2C et SPI, car les capteurs utilisaient différents protocoles. J'ai travaillé sur la transmission de l'information entre les 5 capteurs (luminosité, proximité, vitesse du vents, humidité et gyroscope) simultanément via un bus CAN et l'affichage des données sur une interface en temps réel.
